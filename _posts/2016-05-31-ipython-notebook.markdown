## Math Equation in Ipython Notebook

To add latex equation in Ipython Notebook
```latex
$$c = \sqrt{a^2 + b^2}$$
```

Once run this will render math equation.


## Ploting figure inline

To display matplotlib plots inline on Ipython notebook

```ipython
%matplotlib inline
```
